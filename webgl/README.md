# WebGLによるマンデルブロ集合描画
マンデルブロ集合の描画 with WebGL

２モード実装してます。

1. WASMと同等な動作をするモード
2. 拡大動作をアニメーションでみるモード

# ビルド

## ビルド環境の要件

Node, npm が使えること

## How to ビルド

```sh
npm run build
npm run build2  # アニメーションでみるモード
```

Webサーバで out/, out_anime/ を公開すればOK。

# Notice

## ループMaxについて
「ループMax」に、フラグメントシェーダの `LOOP_MAX` 以上の数値を指定するのは無効です。
インプットタグの挙動上は指定できているように見えたとしても、フラグメントシェーダの実装上の制約によって、 `LOOP_MAX` 値が優先します。

## アニメーションでみるモード
PC(マウスポインタがあること)を想定しています。
キャンバスをクリックすると動作が始まり、もう一度クリックすると一時停止します。
アニメーションはマウスポインタの位置を中心に拡大するように移動します。

拡大率の下限まで行くと止まります。(実装上の制限で、あまり拡大しても粗が見えてしまうので)
